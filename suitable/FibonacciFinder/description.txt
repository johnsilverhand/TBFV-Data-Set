FibonacciFinder-求斐波那契数列第result项恰好大于n

S =
n ≤ 0
⟹ result = 1
∨
n > 0
⟹ ((1 + sqrt(5)) / 2)^{result - 1} - ((1 - sqrt(5)) / 2)^{result - 1} ≤ sqrt(5)n
∧ ((1 + sqrt(5)) / 2)^{result} - ((1 - sqrt(5)) / 2)^{result} > sqrt(5)n



T1: n ≤ 0
D1: result = 1

T2: n > 0
D2: ((1 + sqrt(5))/2)^(result - 1) - ((1 - sqrt(5))/2)^(result - 1) ≤ sqrt(5)n
∧ ((1 + sqrt(5))/2)^result - ((1 - sqrt(5))/2)^result > sqrt(5)n
